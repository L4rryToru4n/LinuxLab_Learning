SSH Installation on Windows 10

If you prefer GUI tools, head to the Installing PuTTY section. Otherwise, head to the Installing OpenSSH section.

Installing PuTTY

Head to the PuTTY installation web page here to download the program. Choose the package files with 32-bit x86 architecture and .msi extension. Or download the msi package directly here

After the download is completed, double click the program with ".msi" extension to run the installation

A welcome window should be shown. Click "Next" to proceed to the next step

On "Destination Folder", you can change the installation destination as you wish then click "Next". If you want to keep the default, Click "Next" again to proceed to the next step

On "Product Features", you can select to add shortcut features or not. Then click "Install".

Wait for installation to finish. If the installation is successfull then click "Finish" to finish the installation.

Installing OpenSSH

To install OpenSSH, open Settings window then head to System > Optional Features. Then click "Add a feature"

Next, type "OpenSSH Client" then check the checkbox feature and click "Install (1)"

Wait for the installation to finished. If successfull, the "Latest actions" section should display the feature as installed.

SSH Installation on Linux

If you have Debian based distro, head to the Installing OpenSSH on Debian based section. For CentOS, head to the Installing OpenSSH on CentOS section. And for Arch based, head to the Installing OpenSSH on Arch section.

Installing OpenSSH on Debian based

To install, type the commands below:

  • sudo apt update
  • sudo apt install openssh-client

Installing OpenSSH on CentOS

To install, type the commands below:

  • sudo dnf install openssh-client

Installing OpenSSH on Arch

To install, type the commands below:

  • sudo pacman -S openssh
