SSH Installation

All of the games that can be played here require you to type texts (command lines) into the terminal / command prompt.

If you already have OpenSSH installed or any similar SSH programs like PuTTY, you can begin playing by clicking Levels. Otherwise, keep reading the guide below to install one.

If your device's OS is Windows and doesn't have an SSH program installed yet, then click here to read the guide on how to install an SSH program into your device.

If your device's OS is UNIX based or Linux and doesn't have an SSH program installed prior to your first OS installation, click here to read the guide on how to install an SSH program for UNIX based OS or Linux. Otherwise, your device should already have an SSH program and you can begin playing by clicking Levels.

Game Guide

Using Terminal / Command Prompt

To connect to the selected level's terminal, simply connect via SSH protocol using either your computer's terminal (Unix/Linux based OS) or command prompt (Windows based OS) and enter the destination address, the port and the password.

For example, ssh linuxlab0@linuxlab.id -p 22 then enter the password to login.

Using PuTTY

On the start of the program, input the hostname destination on the "Host Name (or IP address)" and port number on the "Port" input box. After inputting the hostname destination and port, click "Open" to open a connection to the remote machine

Next, there will be a separate terminal window. On this window enter the username for the current level that you wish to access. For example. linuxlab0, then press Enter.

The remote machine will ask for the username's password. Enter the password for the subsequent level.

If successfull, the terminal screen should print the welcome screen of the remote machine.
